# Landing Page

Static landing page for [slack-chat-migrator](https://github.com/nicklamont/slack-chat-migrator), built with Astro v5, React islands, and Tailwind CSS 4.

Deployed to GitHub Pages via the `deploy-site.yml` workflow on pushes to `main` that touch `site/**`.

## Development

```bash
cd site
npm install
npm run dev       # localhost:4321/slack-chat-migrator
npm run build     # static output in dist/
npm run preview   # preview built site
```
